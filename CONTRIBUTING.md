## 🚀 参与开发

### 相关版本

- Qt 6.6.3
- MSVC 2019 64bit

### 项目结构

```
3rdparty                # 第三方库
└─xxx
    ├─include           # 头文件
    ├─bin               # 可执行文件或动态库（运行时依赖）
    └─lib               # 静态/动态链接库
res                     # 资源文件
└─img                   # 图片资源
utils                   # 工具类/辅助函数代码
windows                 # 主窗口模块
└─mainwindow_child
    ├─homepage          # 主页窗口
    └─settingpage       # 设置窗口
```
